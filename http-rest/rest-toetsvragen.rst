   Welke van deze methods halen gegevens van de server?
   
   * GET
   
     * OK. Met GET haal je het document bij de gegeven URL op
   * POST
   
     * Met POST stuur je een ingevuld formulier naar de server
   * PUT
   
     * Met PUT stuur je een document naar de server
   * HEAD
   
     * OK. Met HEAD haal je de headers op van de gegeven URL
   * DELETE
   
     * DELETE verwijdert een element (resource) van de server
